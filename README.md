# Platform Bengali — ভাষাস্কোপ

**Tracing Language Evolution and Cultural Practice in Digital Economy**

A grant research project by the India Foundation for the Arts (IFA) under its Arts Research programme, investigating Bengali language practices across digital platforms.

## Site Structure

```
├── index.html                    ← Landing page (3D rotating ক, orbital navigation)
├── project-vision.html           ← Project vision & methodology
├── about-ifa.html                ← India Foundation for the Arts
├── resources.html                ← Research materials & references
├── bhashascope.html              ← Project Bhashascope (ভাষাস্কোপ)
├── certificates.html             ← Workshop & event certificates
└── events/
    ├── orientation-workshop.html ← One-day orientation
    ├── mt-workshop.html          ← Five-day MT workshop
    ├── conference.html           ← Hybrid international conference
    ├── youtuber-interviews.html  ← 10 creator interviews
    └── monthly-talks.html        ← 12-month speaker series
```

## Deploy

Connect this repository to [Vercel](https://vercel.com) for instant deployment — no build step required (static HTML).
